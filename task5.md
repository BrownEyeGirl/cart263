# TASK 5 - Project Reflections 
*By Skyla Trousdale*

[Super Real Game:](https://rocmazz.github.io/cart263/topics/project1/) 
	Game is very clever and original, I like the simple layout, its very easy to learn how to play without any guidance despire the unique game flow. I like the personality behind the game narration, taunting you to continue to attempt to click the ‘start’ button. The sarcastic remarks are very clever and make the game feel alive. Though the code is simple and compact, it is concise and well commented. 10/10 would play again, it reminds me of the games I grew up playing on the Bored Button website. I appreciate font and the variation of dom elements used, utilizing many concepts from class so efficiently. Beautiful work. 
	
[You Are Being Optimized](https://namikoko.github.io/project1/)
	From visuals and interaction to narrative and sound, this project covers everything. The experience is beautiful, I was immediately impressed by the quality of the project and the stylistic consistency: I can confidently say that  I have never seen anything like this. The machine interacts with you, and it feels very customized and complete. This looks like a very advanced and stylistically cohesive version of something I’d see on CodePen, but impressively executed. The code lacks comments, which makes me still confused about how exactly the programmer achieved this, but the code is still impressive. I enjoy the ascii stylistic components in the code, and from what I can understand from skimming it, the variables are well named and everything is very well organized. All done with one HTML file, the code is very compact and lovely. I get a lot of joy from interacting with this piece, I love how the narrative was crafted and how almost every single element is animated and loads smoothly. 

[Skinners Box:](https://xp30n.github.io/CART263/PROJ-1/)
 	This project was very visually compelling and interesting. It was haunting being able to operantly condition a poor little creature in Skinner’s Box. The colours were beautiful and followed design principles well, it felt like a professional videogame. The drawings conveyed emotion well and created an attachment with the interface and the creature trapped within it. The code was well commented and explained the functions well, I felt like I could make changes without any confusion. I like that they went above and beyond by implementing a concept and hand drawing the features, as well as unique animation. I would like to see a different font and perhaps more clear gameplay, as it took me a minute to figure out how to use the website. I really enjoyed ths project though and think the concept is very unique.
